# frontend

パッタン（vocab-snap）のフロントエンド。Next.js (App Router) + TypeScript + Tailwind CSS v4。

プロジェクト全体の説明はリポジトリルートの [README](../README.md) を参照。

## セットアップ

```bash
npm install
```

## 開発サーバ起動

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開いて確認。

## ビルド・本番起動

```bash
npm run build
npm start
```

## Lint

```bash
npm run lint
```
